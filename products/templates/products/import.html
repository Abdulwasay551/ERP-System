Supported formats: CSV, Excel (.xlsx, .xls)

File Format Requirements

Required Columns:

  • name - Product name
  • sku - Product SKU (must be unique)
  • product_type - Product type (product, service, etc.)

Optional Columns:

  • description - Product description
  • category - Category name
  • cost_price - Cost price
  • selling_price - Selling price
  • is_active - true/false
  • is_saleable - true/false
  • is_purchasable - true/false
  • is_stockable - true/false

Import Tips:

  • Ensure SKUs are unique across all products
  • Categories will be created if they don't exist
  • Boolean fields accept: true/false, 1/0, yes/no
  • Large files may take several minutes to process
